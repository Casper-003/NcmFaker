# Aura Audio (原 NcmFaker)

一款基于 WinUI 3 构建的现代化、高性能本地音频处理工具。

---

## ✨ 功能特性

- **极速转换**：基于多线程并发技术，快速解密并转换 NCM 格式音频。
- **智能匹配**：内置双搜索引擎，自动获取高精度 Lrc 歌词及专辑封面。
- **元数据内嵌**：支持将歌词及封面直接写入音频文件（ID3 标签），实现无缝播放体验。
- **现代交互**：深度适配 Windows 11 设计语言，支持 Mica（云母）视觉效果及深色模式。
- **自定义配置**：支持任务并发数调节及灵活的输出路径管理。

## 🛠️ 开源组件致谢

本项目的开发离不开以下优秀开源项目的支持：

- [libncmdump](https://github.com/taurusxin/ncmdump)：强大的 NCM 文件解密核心。
- [TagLib-sharp](https://github.com/mono/taglib-sharp)：专业的音频元数据读写库。
- [Windows App SDK (WinUI 3)](https://github.com/microsoft/microsoft-ui-xaml)：提供现代化的 UI 框架支持。
